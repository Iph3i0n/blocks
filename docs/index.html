Documentation Welcome to Starter Dough Get started with this simple tool to build fast and efficient web applications that are beautiful and easy to use. Drop in style! Simply load in the script and get started. All components work out of the box with no external dependencies and will work with any framework or rendering engine. Your favourite, but better Components are built around standard design principles. All of the styles that you are familiar with, but in a new easy format. Web components make it easy to add styles and behaviour in one tag. Say goodbye to forgetting that one class or attribute, or ommitting that tag, that was required to make your page work. Custom themes as standard Custom themes are supported out of the box. Simply put a " Theme " function at the top of the your page and watch the magic happen! Getted started is simple <script src="https://cdn.jsdelivr.net/npm/starter-doug/dist/bundle.js"></script> And you are good to go.
